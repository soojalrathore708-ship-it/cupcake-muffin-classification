# Muffin vs Cupcake Classification 

This project is based on classifying recipes into Muffin or Cupcake using Machine Learning models.

##  Dataset

I created a small dataset of 10 recipes using two features:

* Flour
* Sugar

Each recipe is labeled as Muffin or Cupcake.

##  Exploratory Data Analysis (EDA)

* Checked dataset shape and structure
* Viewed basic statistics
* Visualized data using scatter plot

##  Models Used

* Support Vector Machine (SVM)
* Decision Tree
* K-Nearest Neighbors (KNN)

##  Evaluation

For each model:

* Confusion Matrix
* Classification Report

##  Prediction

A new recipe (Flour = 50, Sugar = 20) was tested and classified using all models.

##  Tools & Libraries

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn
